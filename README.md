程序详细功能介绍
TaskEnder 是一个Windows平台下的窗口进程管理工具，为每个应用程序窗口提供悬浮按钮，快速结束进程或访问系统菜单。主要特性：

悬浮按钮：在每个窗口右上角显示 ⚙️（设置）和 ❌（结束进程）按钮，仅当前活动窗口或未响应窗口显示，避免干扰。

右侧设置侧边栏：点击 ⚙️ 按钮滑出美观的设置面板，包含三大功能：

系统设置：打开 Windows 系统设置。

程序菜单：模拟键盘菜单键（VK_APPS），打开当前窗口的系统菜单（移动、大小、关闭等）。

软件设置：进入详细配置子菜单，可调整：

开机自启动

四种显示模式（控制结束按钮的显示逻辑）

结束进程不询问（慎用）

全屏应用非卡死时隐藏结束按钮

字体选择（系统所有字体列表 + 字号）

进程结束：点击 ❌ 按钮，默认弹出确认框，结束进程；可在设置中开启“不询问”模式。权限不足时提示。

托盘图标：系统托盘常驻，右键可快速打开设置或退出。

单实例运行：防止重复启动。

配置持久化：所有设置保存在 QSettings（Windows 注册表或 .ini 文件）。
